# IKKI デザインシステム — 案B「ネオ和ポップ」(求職者面)

> 2026-07-22 採択。求職者が触る面(診断/結果/シェア)を若手向けに刷新。
> 企業面(`/company`)は端正・信頼の既存路線(`lib/theme.ts` の C)を維持する。
> 朱印モチーフはアクセントとして継続。

## 原則
- 余白を大胆に取る / タイポ階層を強くつける / 色数を絞る(基調2+アクセント1)
- 数字を主役級に大きく / 意味のない装飾・影・グラデを足さない
- 禁止: 紫グラデ・全部角丸・絵文字多用・汎用AIテンプレ的な見た目
- モバイル縦・片手ファースト(主要操作は下部) / reduced-motionで全アニメ無効
- コントラスト 4.5:1以上 / タップ領域 44px以上 / 初回表示を重くしない

## トークンの単一ソース
`lib/design.ts`(求職者面) にすべて定義し、全画面がこれを参照する。
色・タイポ・余白・角丸・影・モーション(duration/easing)を一元化し、
個別ハードコードを排除する。

## 世界観
花札・江戸の刷り物・銭湯ペンキ絵の「色面」、riso印刷。境界は線でなく
色面の切り替えで作る。診断は「おみくじ/御守りを引く」体験。

## パレット(hex)
- 藍(base1・面) `#123A5A` / 藍濃 `#0E2C46`
- 生成り(base2) `#F2ECDD` / 生成り濃 `#E7DBC0`
- 墨(ink) `#1A1712`
- 朱(accent) `#EB4B2F` / 朱濃(文字用) `#C43A22`
- 補助 muted `#6E655A`(生成り上) / 藍上 `#9DB0C2`
- ヘアライン `#D8CDB4`(生成り上) / 藍上 `rgba(242,236,221,.16)`

コントラスト方針: 本文は 墨/生成り or 生成り/藍(いずれも高コントラスト)。
朱は「面+白の大きな文字(≥18px/太)」か「朱濃で小文字」に限定。

## タイポグラフィ
- 見出し: Zen Old Mincho(明朝)。縦書きを主役に使う
- 本文: IBM Plex Sans JP
- ラベル/数字: IBM Plex Mono(数字はtabular)。数字は主役級に大きく
- 階層を強く: display 40 / h1 30 / h2 22 / body 14-16 / label 11

## 余白・角丸・境界
- 余白は色面のバランスで取る(セクション間に大きな"間")
- 角丸 小(sm4 / md8 / lg12 / pill999)。全部丸めない
- 境界は色面の切り替え優先。線は1pxヘアラインのみ

## モーション
- duration: fast160 / base280 / slow440(ms)
- easing: standard(.2,.8,.2,1) / decel(0,.7,.2,1) / accel(.4,0,1,.4) / stamp(.2,1.4,.4,1)
- 遷移に方向と意味: 前進=右から / 戻る=左から / 階層下降=スケール
- 結果は「色面ワイプ→朱印スタンプ」のおみくじ開封演出
- reduced-motion時は全アニメを即時表示にフォールバック
